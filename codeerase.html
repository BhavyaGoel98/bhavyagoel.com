About CodeErase & Roadmap

The vision and future of the ultimate coding playground.

Our Mission

CodeErase was built to bridge the gap between simple web playgrounds and complex local development environments. We believe developers should be able to write and test any language, instantly, with zero setup, exactly as if it were deployed on a production server.

Product Roadmap

v1

The Foundation (Current)

  • Frontend SPA architecture
  • Real-time HTML/CSS/JS web preview engine
  • Multi-language editor support via CodeMirror
  • Piston API integration for backend language execution
  • Premium Dark UI design system
v2

Advanced Runtimes

  • Real database connections (PostgreSQL, MongoDB)
  • File system access simulation
  • NPM package auto-installation
  • WebAssembly direct execution
v3

Docker Sandboxes

  • Full isolated Docker containers per project
  • Terminal access (bash/zsh)
  • Custom Dockerfile support
  • Multi-container orchestration (Docker Compose)
v4

AI & Collaboration

  • AI Code review and generation
  • Multiplayer real-time collaboration
  • One-click deployment to Vercel/AWS
